1.Insert Sorting - Straight Insertion Sort

Description:

Put one item into a sorted list and and generate a new sorted list with length plus one. Then iterate this process until all the item has been inserted into the list.

We need initialize a new variable(guard) to exchange the value in list when add new item(temporary storage) and do the boundary check.

If we meet a new item which is equale to the element in the sorted list, it does not matter to insert this new item before or after the existed one. And this insert action does not broke the sorting sequence. Therefor, the insert sort is stable.

Implementation

Time complexity

O(n^2)

2.Insert Sorting - Shell's Sorting

Description:

Shellsort, also known as Shell sort or Shell's method, is an in-place comparison sort.

In shell sort, we first set a key and broke down the list into some set by this key. After that we sort these small list and reduce the value of key and iterated this process.

How to operate:

  1. Set a key list t1, t2, t3....tk. Make sure that ti > tj(i>j) and tk = 1
  2. Sort this list by this key

Implementation

Decalare that d = {n/2, n/4, n/8...1}
